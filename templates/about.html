Tech Stack

  • Frontend: HTML, CSS, vanilla JS
  • Backend: FastAPI with Jinja templates
  • Data: JSON (extracted / fallback from AQA Paper 3 MCQs)
  • Charts: Chart.js for simple visualizations

Adaptive Engine (Simplified)

We track your level (1–10) and streak. Correct answers raise the streak; every 3 in a row increase your level. Wrong answers reset the streak and can reduce level. Levels bias questions: levels 1–3 → easy, 4–6 → medium, 7–10 → hard.

Notes

  • PDF parsing is naive; we ship fallback, hand-checked items with correct answers.
  • Per-session progress is stored in-memory using a signed cookie for a lightweight demo.
